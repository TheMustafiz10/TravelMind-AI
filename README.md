# TravelMind-AI
A multi-agent AI Travel Planner built with CrewAI that intelligently coordinates multiple specialized agents to create personalized travel itineraries. It leverages the Serper Dev API for real-time web search to gather up-to-date information on destinations, accommodations, and activities, while integrating the Groq Free API for fast and efficient LLM-powered reasoning and decision-making. The system simulates collaborative planning by assigning roles such as researcher, itinerary builder, and optimizer, enabling dynamic, context-aware travel plans tailored to user preferences, budget, and constraints. This project demonstrates the power of agent-based AI systems in automating complex planning tasks with scalability and flexibility.
